# laundry-services
## How to Run the Project

1. Download or clone the repository.
2. Place all project files in the same folder.
3. Open the folder in any code editor such as VS Code.
4. Run the `index.html` file using a web browser.
5. The Laundry Service Login Page will open and display the login interface.

---

## Purpose of the Project
The purpose of this project is to practice:
- HTML page structure
- CSS styling
- Form design
- Layout alignment
- External stylesheet linking

## Output
The webpage contains:
- Laundry service heading
- Login form
- Username input field
- Password input field
