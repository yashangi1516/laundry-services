# Laundry Service Login Page

## Project Description
This project is a simple Laundry Service Login Page created using HTML and CSS.  
The webpage provides a clean and responsive user interface for users to log in to the laundry service platform.

---

## Technologies Used

- HTML5
- CSS3

---

## Files Included

- `index.html`
- `style.css`
- `README.md`

---

## Features

- User-friendly login interface
- Responsive webpage layout
- Styled input fields and login button
- Clean and modern design
- External CSS styling

---

## HTML Features

- Proper HTML document structure
- Login form using input fields
- Button for login submission
- Semantic and organized code

---

## CSS Features

- External stylesheet used
- Custom button styling
- Background colors and alignment
- Margin and padding adjustments
- Responsive sizing using viewport units
